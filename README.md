# 🎮 Funny Puzzle - Java Game

A desktop-based 4×4 sliding puzzle game developed using **Java Swing** and **MySQL** as part of an Object-Oriented Programming (OOP) coursework project.

## ✨ Features

- Interactive Java Swing graphical user interface
- 4×4 sliding puzzle gameplay
- Player login
- Countdown timer
- Move counter
- Pause, Restart, and Reset options
- MySQL-powered leaderboard
- Background music and sound effects

## 🛠️ Technologies Used

- Java
- Java Swing
- MySQL
- JDBC
- Object-Oriented Programming (OOP)
- NetBeans IDE

## 🚀 Getting Started

1. Clone the repository.
2. Open the project in NetBeans IDE.
3. Create the MySQL database and update the database connection if required.
4. Run the project.
